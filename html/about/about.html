About the Company

Our company has three main users. They are the employee, company and the system admin. Employees can apply for relevant job vacancies and different companies can also register in our website. So we can manage the job applications and select the relevant applicants. And also the system admin can control the web system and can see who has registered in the system as an employee or company using the admin page. When a company request to register in the web site system admin will see the request of the company as a disabled account admin and can enable the account by checking details of the company. The system admin can enable or disable any of the accounts that are related to employees or companies. Companies can come to the website and check the website. If they need to post vacancies on website, they should register on the website using register page as a company.
